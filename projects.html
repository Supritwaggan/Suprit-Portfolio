APNA PG

  • HTML/CSS, PHP, Javascript, MYSql, Bootstrap

Apna PG is a web-based application designed to streamline the process of finding and managing Paying Guest accommodations. The project provides a user-friendly interface where PG owners can list their properties, and potential tenants can search for accommodations based on location, amenities, and budget. The system includes features such as booking management, reviews, and a secure payment gateway to facilitate transactions. Apna PG aims to bridge the gap between PG owners and tenants by offering a seamless and efficient platform for both parties. This project was developed using HTML, CSS, JavaScript, and MySQL for the database.

WORKERS PARADISE

  • HTML/CSS, Bootstrap, Javascript, Mysql

Worker's Paradise is a comprehensive platform designed to connect skilled laborers with potential employers or clients looking for specific services. This project offers a user-friendly interface where workers can create profiles highlighting their skills and experience, while employers can post job listings and search for workers based on specific criteria. Key features include real-time job notifications, secure messaging, and a rating system to ensure quality and reliability. Developed using Java, HTML, CSS, and MySQL, Worker's Paradise aims to create an efficient and reliable marketplace for both workers and employers, enhancing job opportunities and service accessibility.
